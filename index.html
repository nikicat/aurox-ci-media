aurox screencasts

CI artifacts for aurox.

main/ — base recordings. Refreshed by the Screencasts workflow on each merge to aurox's main branch; PRs are compared against it. Empty until the first screencast-carrying PR merges — so the compare/diff views for the very first PR correctly show “no base on main yet”.

pr-<N>/ — per-PR recordings. Linked from each PR's screencasts status check, which also links the base-vs-PR side-by-side player (compare.html?pr=<N>, one shared scrub bar) and the base-vs-PR text diff (diff.html?pr=<N>, normalized transcript).
